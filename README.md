# Practice Portfolio Website

This is a personal portfolio website I built to **practice Tailwind CSS, responsive design, and front-end development concepts**. It’s designed to showcase my projects, skills, and contact links while experimenting with modern layouts, animations, and UI effects.

The site helped me strengthen my understanding of Tailwind utilities, flexbox, grids, responsive breakpoints, and interactive elements.

---

## Live Demo

Check it out here: [https://imdadullah.netlify.app/](https://imdadullah.netlify.app/)

---

## Features

- Fully **responsive layout** using Tailwind CSS and custom CSS  
- Smooth **loading screen** with spinner animation  
- **Landing section** with background video and gradient overlay  
- Navigation bar with **desktop and mobile menus**  
- Projects section with clickable links to live demos  
- Skills section with **animated technology icons**  
- Footer with social links: GitHub, LinkedIn, Discord, and Upwork  
- Interactive buttons with hover effects and transitions  
- Gradient overlays and subtle animations for a modern feel  

---

## Technologies Used

- **HTML5** – Semantic structure  
- **Tailwind CSS** – Layouts, utilities, responsive design  
- **JavaScript** – Loading screen logic and menu toggle  
- **Font Awesome** – Social icons  
- **Bootstrap** – For testing layouts and components (optional)  
- **Background video & gradients** – Dynamic landing experience  

---

## Learning Goals

This project was primarily for **practice and experimentation**:

- Applying Tailwind CSS utilities for layouts and spacing  
- Practicing responsive design across multiple breakpoints  
- Creating smooth transitions, hover effects, and animations  
- Structuring a personal portfolio with sections for projects, skills, and contact  
- Combining Tailwind with custom CSS for fine-tuned styling  

This project helped me solidify my Tailwind workflow and front-end development fundamentals.
